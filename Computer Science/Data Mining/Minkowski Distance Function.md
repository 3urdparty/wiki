$d(i,j)\geq 0$ if $i \neq j$
$d(i,j)= 0$ if $i = j$
$d(i,j)= d(j,i)$ (Symmetrical)
$d(i,j) \leq d(i,k) + d(k,j)$ (Triangle inequality, going directly from object $i$ to object $j$ in space is no more than making a detour over any other object $k$)
